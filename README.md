# vuejs-cms

This repository combines a Vue.js frontend with an Express/MongoDB backend to form a minimal CMS.

## Getting started

See the installation instructions: [INSTALLATION.md](INSTALLATION.md)

## Features

- Public home page listing posts
- Admin CRUD interface for posts (create/edit/delete)
- MongoDB for data persistence, via Mongoose
 - Postgres for data persistence (using `pg`)

## Extending

You can add authentication, support for pages or media, or any other CMS capability. Contributions are welcome!
